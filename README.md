# 💪 Neufit

Aplicativo PWA de treino, emagrecimento e evolução pessoal.

## Funcionalidades

✅ Controle de peso  
✅ Cálculo de IMC  
✅ Treinos personalizados  
✅ Plano alimentar  
✅ Controle de água  
✅ Lembrete de medicamentos  
✅ Sistema de XP e níveis  
✅ Medalhas  
✅ Gráfico de evolução  
✅ Backup dos dados  
✅ Funciona como aplicativo no celular

## Tecnologia

- HTML
- CSS
- JavaScript
- PWA
- LocalStorage

## Objetivo

Ajudar pessoas a criarem uma rotina saudável de treino,
alimentação e acompanhamento de evolução.
